# Trabajo final integrador - MonArgent

Propuesta de software para la Tecnicatura Universitaria en Programación (UTN FRC).

##  Stack tecnológico
- **Backend:** Java, Spring Boot, Spring Security, JWT, Hibernate / JPA.
- **Frontend:** React (Vite), Tailwind CSS, PWA (Progressive Web App).
- **Base de Datos:** MySQL.
- **Integraciones:** Gemini API (Procesamiento Inteligente) y OCR.Space (Lectura de tickets).

##  Gestión del proyecto
- **Metodología:** Scrum (Organizado en 7 Sprints).
- **Seguimiento:** Tablero ágil en Jira.
